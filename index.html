Ballance Massively Multiplayer Online - Download Site

About

The Ballance multiplayer experience, finally around. Install Ballance Massively Multiplayer Online (BallanceMMO / BMMO) Mod and play, together with your friends.

Made by Swung0x48 and BallanceBug. Github repository: Swung0x48/BallanceMMO

Installation

For clients, make sure you have Ballance with Ballance Mod Loader installed, then download the client mod package (with dependencies), and place it into (BallanceDirectory)\ModLoader\Mods\. Unzipping of the package is not required but may be desired if the user wishes to download only the bare mod file.

Download

  • Client Mod Package

    Requires Windows 7+ and Ballance Mod Loader 0.3.43 (download). If you don't know which version to choose, just select and download the standalone package.

    Important notice: If you are updating from 3.5.1 or below then you must remove all .dll files in your Mods directory and download the dependencies again! From 3.5.4 on you also need to update your BallanceModLoader to 0.3.43.
